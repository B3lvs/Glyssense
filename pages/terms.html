Termos de Uso

Leia antes de usar o GlySense

1. Aceitação dos Termos

Ao usar GlySense, você concorda com estes termos. Se não concordar, não use nosso serviço.

2. Descrição do Serviço

GlySense é um sistema de monitoramento contínuo de glicose que consiste em:

  • Sensor vestível de glicose
  • Aplicativo móvel
  • Plataforma web
  • Suporte ao cliente

3. Uso Apropriado

Você concorda em usar GlySense apenas para:

  • ✓ Monitoramento pessoal de glicose
  • ✓ Compartilhamento com profissionais de saúde autorizados
  • ✓ Fins educacionais e de pesquisa (com consentimento)

⚠️ Restrições: Você concorda em NÃO usar GlySense para vender dados, revender sensores, ou usar dados para fins discriminatórios.

4. Responsabilidades do Usuário

  • Fornecer informações precisas na criação de conta
  • Manter senha segura e confidencial
  • Reportar uso não autorizado imediatamente
  • Não compartilhar sua conta com terceiros
  • Seguir instruções de aplicação do sensor
  • Consultar médico para interpretação de dados

5. Isenção de Responsabilidade Médica

IMPORTANTE:

  • GlySense não é substituto para cuidado médico profissional
  • Sempre consulte seu médico sobre suas leituras
  • Em emergências, ligue 192 ou procure hospital

  • Você é responsável por decisões de saúde baseadas em nossos dados

6. Precisão de Dados

GlySense oferece precisão de ±15% conforme padrão ISO 15197:2013. Isso significa:

  • Nem sempre 100% acurado
  • Compare com glicômetro tradicional quando em dúvida
  • Use principalmente para tendências, não valores isolados

GlySense não é responsável por:

  • Indisponibilidade temporária do serviço
  • Danos causados por força maior

8. Garantia Limitada
